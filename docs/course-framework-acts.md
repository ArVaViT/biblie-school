# ЧАСТЬ I. ПЕДАГОГИЧЕСКАЯ МЕТОДОЛОГИЯ

## 1.1 Принципы андрагогики (Малколм Ноулз)

Взрослые учатся иначе, чем дети. Курс строится на шести принципах андрагогики:

| Принцип | Реализация в курсе |
|---|---|
| **Потребность в знании** | Каждый модуль начинается с провокационного вопроса: «Почему церковь выжила после гонений?», «Что изменил Совет в Иерусалиме для нас сегодня?» — студент понимает *зачем* ему этот материал |
| **Самоконцепция обучающегося** | Студент выбирает темп прохождения; рефлексивные задания связывают текст с его собственным опытом веры |
| **Роль опыта** | Практические задания опираются на жизненный и церковный опыт студента: «Опишите ситуацию из вашей общины, похожую на конфликт Павла и Варнавы» |
| **Готовность к обучению** | Материал привязан к практическим задачам: проповедь, служение, миссия, лидерство |
| **Ориентация на проблему** | Case-study подход: каждый модуль разбирает реальную проблему ранней церкви как кейс для решения |
| **Мотивация** | Баланс внутренней мотивации (духовный рост) и внешней (сертификат, прогресс-бар, бейджи за модули) |

## 1.2 Таксономия Блума: лестница целей обучения

Курс выстроен по восходящей траектории когнитивных уровней:

```
Модули 1–3   → ЗАПОМНИТЬ / ПОНЯТЬ      (знание фактов, пересказ событий)
Модули 4–6   → ПРИМЕНИТЬ / АНАЛИЗИРОВАТЬ (сравнение, выявление паттернов)
Модули 7–9   → ОЦЕНИТЬ                  (богословская оценка, критический анализ)
Модуль 10    → СОЗДАТЬ                  (авторский проект: проповедь / эссе / план миссии)
```

Примеры learning objectives по уровням:

| Уровень | Глагол | Пример цели |
|---|---|---|
| Запомнить | перечислить, назвать | Перечислить три миссионерских путешествия Павла и их географию |
| Понять | объяснить, описать | Объяснить значение Пятидесятницы для формирования церкви |
| Применить | продемонстрировать, использовать | Использовать принципы решения конфликтов из Деян. 15 в современном церковном контексте |
| Анализировать | сравнить, классифицировать | Сравнить проповеди Петра (Деян. 2) и Павла (Деян. 17) по структуре и аудитории |
| Оценить | обосновать, критически оценить | Обосновать богословское значение включения язычников для экклезиологии |
| Создать | разработать, составить | Разработать план миссионерской стратегии на основе модели Павла |

## 1.4 Принципы микрообучения (Microlearning)

Для онлайн-формата критически важно разбивать материал на малые порции:

- **Один урок (глава) = 15–25 минут** активного обучения
- **Один квиз = 5–8 вопросов** (не более 10 минут)
- **Один ключевой концепт на урок** — не смешивать несколько тем
- **Формат «бутерброд»**: теория (текст/видео 7–10 мин) → вопрос для размышления → практика (квиз или мини-задание)
- **Мобильная адаптация**: весь контент доступен с телефона, видео — не длиннее 12 минут

## 1.5 Scaffolding — постепенное усложнение

```
Уровень 1 (Модули 1–3):  Высокая поддержка
  → Подробные тексты с пояснениями
  → Квизы с подсказками («Перечитайте Деян. 2:1-4»)
  → Рефлексии с направляющими вопросами

Уровень 2 (Модули 4–7):  Средняя поддержка
  → Тексты требуют самостоятельного анализа
  → Квизы без подсказок
  → Эссе с чёткими критериями, но без шаблона

Уровень 3 (Модули 8–10): Минимальная поддержка
  → Студент самостоятельно находит связи между текстами
  → Критическое эссе или авторский проект
  → Peer-review (взаимная оценка)
```

## 1.6 Активное обучение vs пассивное потребление

| Пассивные методы (≤30% времени) | Активные методы (≥70% времени) |
|---|---|
| Чтение лекционного текста | Вопросы для размышления после каждого раздела |
| Просмотр видео | Квизы с немедленной обратной связью |
| Прослушивание аудио-комментария | Рефлексивное эссе: «Как этот текст говорит в мою ситуацию?» |
| | Сравнительный анализ отрывков |
| | Составление собственной хронологии / карты |
| | Групповая дискуссия (форум) |
| | Итоговый творческий проект |

## Модуль 1. Введение: контекст и замысел Луки

**Деяния 1:1-11**

### Learning Outcomes
1. **Запомнить**: Назвать автора, адресата и жанр книги Деяний; определить хронологические рамки повествования
2. **Понять**: Объяснить связь между Евангелием от Луки и Деяниями как двухтомным произведением
3. **Понять**: Описать программный текст Деян. 1:8 как тематический план всей книги

### Структура уроков

| Урок | Тема | Тип контента | Время |
|---|---|---|---|
| 1.1 | Кто написал Деяния и зачем? Автор, жанр, датировка | Текст + инфографика | 60 мин |
| 1.2 | Лука-Деяния: два тома одной истории | Видео-лекция (10 мин) + текст | 70 мин |
| 1.3 | Деяния 1:8 — программный стих: «от Иерусалима до края земли» | Текст + интерактивная карта | 60 мин |
| 1.4 | Вознесение и обетование возвращения (Деян. 1:9-11) | Текст + рефлексия | 50 мин |

### Оценивание
- **Формативное**: Квиз после каждого урока (5–7 вопросов, множественный выбор + верно/неверно)
- **Суммативное**: Мини-эссе (300–500 слов): «Почему Лука написал второй том? Какова его цель для первых читателей и для нас?»

### Контент каждого урока (шаблон «бутерброд»)
1. Текст лекции / комментарий (1000–1500 слов)
2. 2–3 вопроса для размышления (встроены в текст)
3. Квиз (5–7 вопросов с автоматической проверкой)
4. Мини-задание: рефлексия / ответ на форуме

# ЧАСТЬ IV. ПСИХОЛОГИЯ ОБУЧЕНИЯ

## 4.1 Кривая забывания Эббингауза и интервальное повторение

Без повторения студент забывает ~70% материала через 24 часа. Стратегии противодействия:

| Стратегия | Реализация в курсе |
|---|---|
| **Квиз сразу после урока** | Первый интервал: немедленное повторение ключевых фактов |
| **Вопросы-якоря в следующих уроках** | Каждый урок начинается с 1–2 вопросов по предыдущему материалу |
| **Кумулятивные квизы** | Квиз модуля 5 включает 20% вопросов из модулей 1–4 |
| **Финальный обзорный урок (10.4)** | Систематический обзор всех ключевых тем |
| **Итоговый проект** | Требует синтеза материала всего курса — глубокое повторение через применение |

### Модель интервального повторения в квизах

```
Урок 3.2:
  → Квиз 3.2 — 100% новый материал
  
Урок 3.3:
  → Квиз 3.3 — 80% новый + 20% из урока 3.1–3.2
  
Модуль 4, Урок 4.1:
  → Квиз 4.1 — 80% новый + 10% из Модуля 3 + 10% из Модулей 1–2
```

## 4.2 Мотивация: внутренняя и внешняя

### Внутренняя мотивация (приоритет)

| Фактор | Реализация |
|---|---|
| **Автономия** | Выбор формата итогового проекта; выбор темпа прохождения; возможность пропустить модуль при наличии предзнаний (входной тест) |
| **Компетентность** | Квизы с немедленной обратной связью; прогресс-бар; бейджи за завершение модулей |
| **Связь с другими** | Форумные обсуждения; групповые задания; peer-review; сообщество курса |
| **Смысл** | Постоянная связь с практикой: «как это влияет на вашу веру, служение, общину?» |

### Внешняя мотивация (поддерживающая)

- Сертификат по завершении курса
- Прогресс-бар и визуальное отслеживание прохождения
- Бейджи за завершение модулей (элементы геймификации)
- Напоминания по email при длительном отсутствии

## 4.3 Эмоциональное вовлечение через нарратив

Книга Деяний — уже нарратив. Курс усиливает это:

- **Сквозной сюжет**: курс подаётся как «путешествие» от Иерусалима до Рима вместе с героями
- **Персонализация**: «Представьте, что вы — один из 120 в горнице. Что вы чувствуете?»
- **Клиффхэнгеры между модулями**: «Савл только что одобрил убийство Стефана. Что будет дальше? Узнаете в Модуле 5»
- **Драматургия**: конфликт (гонения), напряжение (суды Павла), развязка (Рим)
- **Отождествление**: задания на рефлексию: «С каким персонажем Деяний вы себя отождествляете и почему?»

## 4.4 Теория когнитивной нагрузки (Sweller)

| Тип нагрузки | Проблема | Решение в курсе |
|---|---|---|
| **Внутренняя** (сложность материала) | Древний контекст, незнакомая география, богословские концепции | Scaffolding: глоссарий, карты, хронологии; постепенное усложнение |
| **Внешняя** (плохой дизайн) | Перегруженные страницы, запутанная навигация | Чистый дизайн LMS; один концепт на экран; навигация «хлебные крошки» |
| **Germane** (полезная нагрузка) | Работа над пониманием и интеграцией знаний | Вопросы для размышления, сравнительные таблицы, рефлексии |

### Практические правила:

- **Один урок = один главный концепт** (не более)
- **Текст лекции: 1000–1500 слов** (не более 2000)
- **Видео: 7–12 минут** (разбивать длинные на части)
- **Квиз: 5–8 вопросов** (не более 10)
- **Визуальные якоря**: карты, хронологии, схемы разбивают текст и снижают нагрузку
- **Белое пространство**: не загромождать экран

## 4.5 Принципы мультимедийного обучения (Mayer)

| Принцип | Описание | Реализация |
|---|---|---|
| **Мультимедиа** | Текст + графика лучше, чем только текст | Карты, схемы, инфографика в каждом модуле |
| **Смежность** | Текст и изображение рядом, не на разных страницах | Карта маршрута — рядом с описанием путешествия |
| **Модальность** | Анимация + аудио лучше, чем анимация + текст на экране | Видео-лекции: говорящий преподаватель + визуальный ряд, минимум текста на экране |
| **Избыточность** | Не дублировать аудио текстом на экране | Видео не дублирует текст лекции — они дополняют друг друга |
| **Сегментация** | Разбивать на управляемые порции | Микрообучение: 15–25 мин на урок |
| **Персонализация** | Разговорный стиль вовлекает больше | Обращение к студенту: «вы», «подумайте», «представьте» |
| **Сигнализация** | Выделять ключевые моменты | Жирный шрифт, выноски «Ключевой момент:», маркированные списки |

## 6.3 Спецификация типов вопросов для квизов

### Множественный выбор (60% вопросов)
- 4 варианта ответа (1 правильный)
- Дистракторы правдоподобны, но отличимы при знании текста
- Обратная связь объясняет, почему правильный ответ верен

### Верно / Неверно (20% вопросов)
- Утверждения проверяют распространённые заблуждения
- Обратная связь содержит ссылку на текст Деяний

### Соотнесение (10% вопросов)
- Соотнести персонажа с событием, город с путешествием, цитату с автором

### Краткий ответ (10% вопросов)
- Проверяет запоминание ключевых фактов
- Допускает вариации формулировок

## 6.4 Спецификация практических заданий

### Рефлексия (после каждого модуля)
- Объём: 150–300 слов
- Направляющий вопрос: «Что в этом модуле затронуло вас лично? Как это влияет на вашу веру/служение?»
- Оценка: зачёт/незачёт (критерий — содержательность ответа)

### Мини-эссе (модули 1, 4)
- Объём: 300–500 слов
- Конкретный вопрос, требующий анализа одного аспекта
- Оценка по рубрике (упрощённой)

### Аналитическое эссе (модули 3, 5, 7, 8, 9)
- Объём: 500–1000 слов
- Требует сравнения, критической оценки, обоснования позиции
- Оценка по полной рубрике

### Практическое задание (модуль 6, 8)
- Создание прикладного артефакта (план проповеди, стратегия)
- Оценка по критериям практической применимости

### Групповое задание (модуль 7)
- Ролевая дискуссия на форуме
- Оценка: участие + качество аргументации
